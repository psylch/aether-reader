# Phase 3: 页面管理

独立的页面级 PDF 操作模块，不依赖标注或编辑功能。

## 扩展点

- 侧边栏缩略图增加页面操作右键菜单
- 菜单栏增加 Pages 菜单
- 新增 PageManagerView (独立页面管理界面)

## 功能清单

### 页面操作
- [ ] 插入页面 (空白页 / 从其他 PDF / 从图片)
- [ ] 删除页面
- [ ] 提取页面为新 PDF
- [ ] 重新排序 (缩略图拖拽)
- [ ] 旋转页面 (单页 / 批量)
- [ ] 裁剪页面 (可视化裁剪框)
- [ ] 复制页面

### 文档操作
- [ ] 合并多个 PDF (拖拽排序 + 页面范围选择)
- [ ] 拆分 PDF (按页码 / 按大小 / 按书签)
- [ ] 压缩 PDF (质量可选: 高/中/低)
- [ ] 展平 PDF (Flatten — 批注/表单固化为静态内容)

### 页面元素
- [ ] 页眉 / 页脚 (自定义内容: 页码/日期/文件名, 可选页面范围)
- [ ] 水印 (文字 / 图片, 可调透明度/角度/位置/页面范围)
- [ ] 背景 (颜色 / 图片)
- [ ] Bates 编号 (前缀 + 起始数字 + 位数)
- [ ] 链接编辑 (添加/修改/删除超链接)

## 实施步骤

### Step 1: 页面操作基础
- PageManagementService (PDFDocument 页面操作封装)
- 缩略图右键菜单 (插入/删除/提取/旋转/复制)
- 缩略图拖拽排序

### Step 2: 裁剪 + 合并 + 拆分
- CropOverlayView (可视化裁剪框)
- MergeView (多文件拖入 + 排序 + 页面范围)
- SplitView (拆分规则配置)

### Step 3: 压缩 + 展平
- 压缩引擎 (图片重采样 + 质量控制)
- 展平实现 (PDFAnnotation → 静态内容)

### Step 4: 页面元素
- HeaderFooterView (配置 + 预览)
- WatermarkView (文字/图片 + 参数调整)
- BackgroundView
- Bates 编号生成器
- 链接编辑工具
