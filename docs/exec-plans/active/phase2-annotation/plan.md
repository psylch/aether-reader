# Phase 2: 标注系统

在 Phase 1 阅读器基础上增加完整的标注/批注能力。

## 扩展点

- ReaderView 工具栏增加标注工具区域
- 侧边栏增加「标注列表」section
- PDFKitView 处理标注交互 (选区 → 标记, 点击 → 放置)

## 功能清单

### 文本标记
- [ ] 高亮 (6 色: 黄/绿/蓝/粉/紫/橙)
- [ ] 区域高亮 (矩形框选)
- [ ] 下划线
- [ ] 删除线
- [ ] 波浪线

### 笔记与文本
- [ ] 便签 (Sticky Note — 点击放置 + 输入文字)
- [ ] 文本框 (Text Box — 自由放置)
- [ ] 引出线标注 (Callout — 箭头 + 文字框)

### 形状
- [ ] 直线
- [ ] 箭头
- [ ] 矩形
- [ ] 椭圆
- [ ] 多边形
- [ ] 云朵形 (Cloud)

### 手绘
- [ ] 墨迹/自由绘制 (可选颜色/粗细)
- [ ] 橡皮擦

### 图章 (Stamps)
- [ ] 内置图章 (Approved / Confidential / Draft / Final 等)
- [ ] 自定义图章 (导入图片)

### 标注管理
- [ ] 标注保存到 PDF (PDFKit 原生写入)
- [ ] 侧边栏标注列表 (按页/按类型/按颜色筛选)
- [ ] 导出标注摘要 (Markdown / TXT)
- [ ] 批量删除标注
- [ ] 标注回复/评论线程

## 实施步骤

### Step 1: 基础架构
- AnnotationTool enum (所有工具类型)
- AnnotationViewModel (@Observable, 管理当前工具/颜色/粗细)
- PDFKitView 标注交互层 (delegate + gesture handling)
- 工具栏标注按钮区域

### Step 2: 文本标记
- 选中文本 → 迷你工具栏 (高亮/下划线/删除线/波浪线)
- 颜色选择器
- 区域高亮 (矩形拖拽)

### Step 3: 便签 + 文本框 + 引出线
- 点击放置 + 编辑弹出
- 文本框自由放置 + 拖拽调整
- 引出线 (箭头起点 + 文字框终点)

### Step 4: 形状 + 手绘
- 形状绘制 (拖拽起止点)
- 墨迹路径收集 + 渲染
- 橡皮擦 (路径级擦除)

### Step 5: 图章 + 管理
- 图章选择面板 + 点击放置
- 自定义图章导入
- 侧边栏标注列表
- 导出标注摘要
